# scarlet-common

Pacchetto condiviso per il sistema Scarlet.

Contiene modelli Pydantic, interfacce base, utility e costanti condivise
da tutti i microservizi.

## Installazione

```bash
pip install -e ./scarlet-common
```

## Contenuto

| Modulo | Descrizione |
|---|---|
| `scarlet_common.logging` | Logger strutturato JSON condiviso |
